## レイヤー構成（推定）

```
レイヤー5: テロップ・テキスト

レイヤー3: アバター（クロマキー合成後）
レイヤー2: 操作画面・スライド
レイヤー1: 背景（3D空間/グラデーション）
```

## 音声構成（推定）

- **ナレーション**: 女性音声、各セクションの説明
- **BGM**: 明るいループ音楽（常時再生）
- **SE**: トランジション時、テキスト表示時に効果音

## アプリ実装への示唆

### 必須機能
1. **レイヤー合成エンジン**: 5層以上のレイヤー管理
2. **クロマキー処理**: VRMアバター動画のグリーンバック除去
3. **テキストオーバーレイ**: 複数スタイルのテロップ配置
4. **トランジション**: フェード、スライド等
5. **パーティクルエフェクト**: キラキラ等のオーバーレイ

### テンプレート化候補
- イントロパターン（アバター＋背景＋テロップ）
- 目次スライドパターン（グラデーション＋テキスト＋ナビキャラ）
- 操作説明パターン（キャプチャ画面＋テロップ＋アバター）
- アウトロパターン（メッセージ＋エフェクト＋キャラ）
